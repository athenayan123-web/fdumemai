# OpenClaw AI 全网比价智能平台

> **复旦大学2025未来信息创新学院工程管理** 出品

## 平台介绍

OpenClaw AI 全网比价智能平台是一个聚焦国际国内 AI 服务实时比价的工具平台，覆盖：

- 📊 **模型API比价**：OpenAI、Anthropic、Google、DeepSeek、Moonshot、阶跃星辰、智谱AI、百度、腾讯混元、字节火山、小米等 19+ 厂商
- 💻 **Coding Plan比价**：GitHub Copilot、Cursor、Windsurf、Augment、Trae、通义灵码等 10+ 产品
- 🖥️ **GPU算力比价**：图灵算力中心实时价格（NVIDIA、沐曦 MetaX）
- 🦞 **Claw集合对比**：OpenClaw、QClaw、KimiClaw、JVSClaw、WorkBuddy、ArkClaw、小米Claw
